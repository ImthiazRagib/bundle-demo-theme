# Shopify Bundle Builder — Theme Integration

Drop these files into any Shopify theme to add a fully functional customizable belt bundle builder.

---

## Folder structure

```
shopify-theme/
├── assets/
│   ├── bundle-builder.css          ← All styles (mobile-first, CSS variables)
│   ├── bundle-builder.js           ← All interactivity (vanilla JS, no framework)
│   ├── bundle-builder-media.json   ← ⚠️ YOUR PHOTO DATABASE — never overwrite after setup
│   └── bundle-builder-catalog.json ← SKU → Variant ID map (generated by Sync Catalog)
├── sections/
│   └── bundle-builder.liquid       ← Main section (6 screens + Shopify schema settings)
├── snippets/
│   └── bundle-cart-item.liquid     ← Cart line-item renderer for bundle line items
└── templates/
    └── page.bundle.liquid          ← Full-page template that renders the builder
```

## Installation steps

### 1. Upload files to your theme

Go to **Shopify Admin → Online Store → Themes → your theme → Edit code**.

| File | Upload to |
|------|-----------|
| `assets/bundle-builder.css` | `assets/` |
| `assets/bundle-builder.js` | `assets/` |
| `assets/bundle-builder-media.json` | `assets/` |
| `assets/bundle-builder-catalog.json` | `assets/` |
| `sections/bundle-builder.liquid` | `sections/` |
| `snippets/bundle-cart-item.liquid` | `snippets/` |
| `templates/page.bundle.liquid` | `templates/` |

---

### 2. Create bundle products in Shopify

Create products (or variants) for each bundle type and note their **Variant IDs**:

| Bundle | Price |
|--------|-------|
| Single Set Belt | €49,99 |
| Double Set Belt | €79,99 |
| Triple Set Belt | €99,99 |
| Infinity Set Belt (base — 4 belts) | €119,99 |
| Infinity Set Belt (extra belt) | €19,99 |

Get the Variant ID from the URL when editing a variant:
`/admin/products/[product-id]/variants/[VARIANT-ID]`

Also create **component products** (set as Draft) tagged `bb-component`:
- **Straps**: one variant per color per length (18 total), SKU format: `STRAP-NERO-130`
- **Buckles**: one per style (4 total), SKU format: `FIBBIA-CLASSIC`
- **Passanti**: one per color (9 total), SKU format: `PASSANTE-NERO`
- **Logistics**: Belt Box (`BOX-CINTURA`), Warranty Card (`WARRANTY-CARD`), NFC Card (`NFC-CARD`)

All component products should be priced at €0 and set to **Draft** so customers cannot find or purchase them directly. They are added to orders automatically for fulfillment traceability.

---

### 3. Create a page

1. **Online Store → Pages → Add page**
2. Name it (e.g. "Crea il tuo set")
3. Set the **Template** to `page.bundle` in the right sidebar
4. Save and publish

---

### 4. Configure via Theme Editor

Go to **Online Store → Themes → Customize → navigate to your bundle page**.

In the **Bundle Builder** section settings, configure:

#### Bundle prices (display text)
| Setting | Example |
|---------|---------|
| Prezzo Single Set Belt | `€ 49,99` |
| Prezzo Double Set Belt | `€ 79,99` |
| Prezzo Triple Set Belt | `€ 99,99` |
| Prezzo Infinity Set Belt | `€ 119,99` |
| Prezzo cintura extra Infinity | `€ 19,99` |

#### Variant IDs — bundle products
Paste the numeric IDs from step 2 into the corresponding fields.

#### Variant IDs — components
Fill in the variant IDs for straps (130cm / 150cm), buckles (1–4), and logistics (warranty, NFC, box).

#### Images
| Setting | Description |
|---------|-------------|
| Immagine Single/Double/Triple/Infinity Set | Card images shown on the selector screen |
| Foto modello | Photo shown in the "Vedi indossata" modal |
| URL video 360° | Direct `.mp4` URL for the 360° belt preview |
| Handle prodotto combinazioni cintura | Shopify product handle for the combo-preview product (variants with SKU + image per combination) |

#### Carousel configuration
- Set the default buckle and strap shown on page load
- Reorder items by editing the comma-separated keys

#### Availability mode
- **Grigio con badge**: greys out out-of-stock items with "Non Disponibile" badge
- **Nascondi**: removes them from the carousel entirely

---

### 5. Add combo photos (optional but recommended)

Edit `assets/bundle-builder-media.json` in the theme editor.

Fill in CDN URLs for:
- `straps.[color].[length]` — strap photos per color × length
- `buckles.[buckle-key]` — buckle thumbnail images
- `combinations.[strap__length__buckle].photo` — specific combo photos
- `combinations.[strap__length__buckle].worn` — "worn on model" photo per combo
- `combinations.[strap__length__buckle].video` — 360° video per combo

> ⚠️ **Never overwrite `bundle-builder-media.json` with a fresh copy from this package** — it will erase all your photo URLs. Only update `.liquid`, `.css`, and `.js` files on theme upgrades.

---

### 6. Integrate bundle display in your cart (optional but recommended)

In your theme's cart template (`sections/main-cart-items.liquid` or `cart.liquid`), wrap your line-item loop to render bundles differently:

```liquid
{% for item in cart.items %}
  {% if item.properties['Cintura 1 - Lunghezza'] != blank %}
    {% render 'bundle-cart-item', item: item %}
  {% else %}
    {# your existing line item template #}
  {% endif %}
{% endfor %}
```

---

### 7. Sync the component catalog

After creating all component products in step 2, run **Sync Catalog** from the admin dashboard. This reads all `bb-component`-tagged products from Shopify, builds the SKU→VariantID map, and writes it to `bundle-builder-catalog.json`.

Without this step the builder still works — it will add only the main bundle product to the cart, without the individual component SKUs for fulfillment.

## How the 6-screen flow works

| Screen | What happens |
|--------|--------------|
| **Selector** | Customer picks a bundle type (Single / Double / Triple / Infinity) |
| **Composer** | Step-by-step: choose Length → Buckle → Strap color for each belt |
| **Review** | Summary of all configured belts with edit buttons |
| **Cart** | Cart with quantity controls, remove, and total |
| **Confirm** | Pre-checkout with trust signals (secure payment, free returns, delivery) |
| **Confirmed** | Thank-you screen (shown after Shopify checkout redirect) |

---

## Cart line item properties

Each bundle is added to the Shopify cart with these visible line item properties:

| Property | Example |
|----------|---------|
| `Cintura 1 - Lunghezza` | `130cm` |
| `Cintura 1 - Fibbia` | `Silver` |
| `Cintura 1 - Colore` | `Nero` |
| `Cintura 2 - Lunghezza` | `150cm` |
| `Cintura 2 - Fibbia` | `Gold` |
| `Cintura 2 - Colore` | `Marrone` |

Hidden properties (prefixed `_`) are also added — visible in Shopify Admin orders for fulfillment but hidden from customer-facing emails.

---

## 3-layer cart structure

Each checkout submission sends:

1. **Bundle product** — the priced, visible line item (e.g. Double Set Belt at €79,99)
2. **Component SKUs at €0** — strap, buckle, passante per belt (for stock tracking)
3. **Logistics SKUs at €0** — belt box, warranty card, NFC card (for fulfillment)

## Customisation

| What | Where |
|------|-------|
| Colors / accent | CSS variables under `.bb-wrap { --bb-accent: ... }` in `bundle-builder.css` |
| Belt lengths | `LENGTHS` array in `bundle-builder.js` |
| Buckle styles | `BUCKLES` object in `bundle-builder.js` |
| Strap colors | `STRAPS` object in `bundle-builder.js` |
| Bundle pricing | Section settings in Theme Editor |
| Carousel order | `buckle_order` / `strap_order` text fields in Theme Editor |
